# Multi-Dimension

## 기획의도
> ### 차원을 초월한 디지털 패턴
```
시작은 점으로 출발했습니다.
점을 재료로 하여 눈에 보이는 선을 만들었고, 그 선은 평면을 깊이 있게 채워나갑니다.

재료는 점이고, 벽지는 평면이지만
우리가 느끼는 감각은 공간이 되었습니다.
```
